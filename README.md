## CRRouter提供了哪些功能

 - 通过一个URL获取一个对象

## 安装
### CocoaPods

1. 在Podfile中添加 pod "CRRouter".
2. 输入命令 `pod install` 或者 `pod update`.
3. Import \<CRRouter/CRRouter.h\>.
